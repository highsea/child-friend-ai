# ESP32-S3 引脚连接文档

## 1. 硬件概述

### 1.1 ESP32-S3 开发板规格

ESP32-S3 Dual Type-C Development Board (N16R8)：
- 模块：ESP32-S3-WROOM-1-N16R8
- 16MB Flash
- 8MB PSRAM
- 内置 2.4GHz WiFi + Bluetooth 5 (LE)
- 板尺寸：57mm × 28mm
- 排针：2×22Pin (44脚)
- 芯片有 45 个可配置 GPIO (GPIO0~GPIO21, GPIO26~GPIO48)，部分引脚被 Flash/PSRAM 占用

## 2. 当前引脚分配

### 2.1 I2S 音频接口 (INMP441 麦克风 + MAX98357 扬声器)

```
│   └────────┬────────┘              └────────┬────────┘             │
│            │                                 │                      │

│            │                                 │                      │
│     WS ────│──► GPIO14 ◄────── LRC ◄────────│                      │

│            │                                 │                      │
│    VDD ───│──► 3.3V                        │                      │
│    GND ───│──► GND                         │                      │

│            │                                 │                      │
│                                        DIN ────│──► GPIO13         │
│                                        GAIN ────│──► GND (默认)    │

             │                                 │
        I2S0 端口                        I2S0 端口 (输出)
```

**代码中的引脚定义** (board.h):

| 功能 | ESP32-S3 GPIO | INMP441 | MAX98357A | 说明 |
|------|---------------|---------|-----------|------|
| BCLK | GPIO12 | SCK | BCK | I2S 时钟 |
| WS | GPIO14 | WS | LRC | 字选通 |
| MIC DIN | GPIO15 | SD | - | 麦克风数据输入 |
| SPK DIN | GPIO13 | - | DIN | 扬声器数据输出 |
| VDD | - | 3.3V | 5V | 电源 |
| GND | - | GND | GND | 地 |
| L/R | - | GND | - | 麦克风左声道 |

**注意**: 当前代码使用**单一 I2S 端口 (I2S_NUM_0)**，同时支持输入和输出。MAX98357A 的 GAIN 引脚接 GND（默认增益 3dB）。

## 5. 注意事项

### 5.1 I2S 资源共享

当前配置使用同一组 GPIO (GPIO14, GPIO15, GPIO35) 兼供麦克风和扬声器：
- I2S0 端口: 麦克风输入
- I2S1 端口: 扬声器输出

这是 **软件时分复用** 方案，理论上可行但需要注意：
1. 同一时刻只能有一个方向工作（采集或播放）
2. 如果需要真正全双工，需要修改硬件连接使用不同的 GPIO

### 5.2 电源要求

| 模块 | 电压 | 电流 |
|------|------|------|
| ESP32-S3 | 3.3V | ~300mA (峰值) |
| INMP441 | 3.3V | ~600µA |
| MAX98357 | 5V (推荐) / 3.3V | ~10mA (静态) |
| ST7735 | 3.3V | ~50mA |

**建议**: 使用 5V 2A 电源适配器，通过 AMS1117-3.3 降压给 ESP32 和屏幕供电。

### 5.3 布线建议

1. **电源线**: 尽量粗短，减少压降
2. **I2S 线**: 麦克风到 ESP32 距离越短越好（<10cm）
3. **SPI 屏幕线**: 可适当加长，但建议 <15cm
4. **地线**: 确保所有模块共地

## 6. 代码中的引脚定义

固件代码中的引脚定义位于：
- `firmware/main/board.h` - I2S 引脚
- `firmware/main/display.c` - SPI 屏幕引脚

如需修改引脚，请编辑这两个文件。
